# Ontario Tax Refund Guide

A simple guided web calculator prototype for estimating common Ontario/Canada tax refunds and benefits.

## Live Page

After GitHub Pages is enabled, the site should be available at:

`https://hongjieli.github.io/ontario-tax-refund-guide/`

## Included

- Guided step-by-step form
- Estimate for common items:
  - possible refund from payroll tax withheld
  - GST/HST Credit
  - Ontario Trillium Benefit
  - Canada Workers Benefit
- Chinese UI
- Disclaimer for rough estimation only

## Disclaimer

This is a rough estimator, not tax filing advice and not an official CRA/Ontario calculator.
